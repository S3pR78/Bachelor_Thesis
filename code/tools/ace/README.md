# ACE Tools

This folder contains helper scripts for working with ACE-related dataset splits, playbooks, and error inspection.

## Scripts

- `create_ace_splits_from_master.py`
  - Create ACE training/validation/test splits from the master dataset.
- `curate_ace_playbook.py`
  - Build or update an ACE playbook with curated examples and metadata.
- `import_llm_deltas_to_playbook.py`
  - Import corrections or delta changes generated by an LLM back into the ACE playbook.
- `inspect_errors_for_ace.py`
  - Analyze ACE model prediction failures and inspect common error patterns.

## Usage

Run scripts from the repository root with `PYTHONPATH=code`, for example:

```bash
PYTHONPATH=code python code/tools/ace/inspect_errors_for_ace.py
```
