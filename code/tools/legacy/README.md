# Legacy Tools

This folder contains legacy or one-off scripts that are no longer part of the main workflow.

## Purpose

Use this directory for historical utilities and experimental scripts that may be useful for reference, but are not intended for active workflow automation.

## Current contents

- `merge.py`
  - Merge script data or outputs from older experiments.

## Note

Prefer using the active workflow directories under `code/tools/` and keep `legacy/` reserved for archival or experimental code.
