# ACE Attribution

Parts of this module are adapted from the official Agentic Context Engineering (ACE) repository:

https://github.com/ace-agent/ace

The original ACE framework introduces the Generator–Reflector–Curator architecture, evolving playbooks, and offline/online/eval-only adaptation modes.

This thesis project adapts the ACE framework to ORKG Text-to-SPARQL generation by adding an ORKG-specific data processor, prompt configuration, and integration with the existing SPARQL/PGMR-lite evaluation pipeline.
